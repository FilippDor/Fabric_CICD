# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

By participating in this project, you agree to abide by its terms.

## Reporting

If you experience or witness unacceptable behavior, please report it by opening a GitHub issue or contacting the maintainer directly.

## Scope

This Code of Conduct applies within all project spaces, including issues, pull requests, discussions, and any other communication channels associated with the project.
